# Klicks To Clients

Klicks To Clients is a premium local growth agency landing page built to turn local visibility into real revenue for service-based businesses in the Greater Tampa Bay area and nationwide.

## Technologies Used

*   **HTML5:** Semantic and accessible structure.
*   **Tailwind CSS:** Utility-first styling (loaded via CDN) for rapid UI development.
*   **Custom Design System:** Configured directly within the Tailwind configuration script (`index.html`) using a modern color palette, custom fonts, and custom animations.
*   **Typography:** Google Fonts ('Plus Jakarta Sans' for display/headings, 'Inter' for body text).
*   **Icons:** Google Material Symbols.

## Features & Highlights

This landing page employs a modern, high-converting aesthetic featuring:

*   **Custom Animations:** Smooth, continuous CSS animations like slow scrolling (`animate-scroll-slow`), pulsing signals (`animate-ping-slow`), and blinking cursors for dynamic terminal/code effects.
*   **Glassmorphism Effects:** Using `backdrop-blur` utilities and translucent custom backgrounds (`glass-card`) for a premium "dashboard" feel.
*   **Interactive UI Elements:** 
    *   FAQ Accordions built using native HTML details/summary elements with custom rotating icons.
    *   Hover states and micro-interactions on cards and buttons to encourage user engagement.
*   **Dark Theme Aesthetics:** The page utilizes a deep, dark color scheme (`#08131F`, `#0B1A2A`) accented by vibrant cyan (`#2ED8C3`), lime/yellow (`#C9F84A`), and blue (`#3B82F6`) to create a striking "tech/growth" aesthetic visually akin to modern SaaS and agency platforms.

## Running Locally

Since this project primarily utilizes a single `index.html` file with a CDN-linked Tailwind script, no complex build steps are necessary to view the layout. 

Simply open `index.html` in any modern web browser.
